# Explore commands
1. `❯ man man`
2. `❯ ls`
3. `❯ cd`
4. `❯ pwd`
5. `❯ mkdir`
6. `❯ rmdir`
7. `❯ touch`
8. `❯ cp`
9. `❯ mv`
10. `❯ rm`
11. `❯ cat /etc/*-release`
12. `❯ less --help`
13. `❯ grep`
14. `❯ find`
15. `❯ chmod`
16. `❯ chown`
17. `❯ sudo`
18. `❯ ps`
19. `❯ kill -l | tr ' ' '\n' | bat`
20. `❯ which ls`
21. `❯ curl https://cht.sh | less`
